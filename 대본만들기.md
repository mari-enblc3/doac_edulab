# 세션 대본(인터뷰 스크립트) 만드는 법

매주 화·수·목 세션 전에, DOAC 영상 하나를 앱에 넣을 "스크립트 JSON"으로 만드는 방법.

## 방법 A — Claude에게 맡기기 (지금 방식, 설치·비용 0)

Claude Code 세션에서 이렇게 요청하세요:

> "이 DOAC 영상 대본을 인터뷰 스크립트 JSON으로 만들어줘: <유튜브 링크>"

Claude가 자막을 가져와서:
- 인물별 대사로 구분 (Steven / Guest 등)
- 각 대사에 한글 번역 병기
- 챕터를 **30분 분량**으로 크게 묶기
- 아래 JSON 포맷으로 출력

받은 JSON을 앱 → **+ 영상 추가 → 스크립트 JSON** 탭에 붙여넣기 → 학습 시작.

## JSON 포맷

```json
{
  "title": "에피소드 제목",
  "youtubeId": "영상ID 또는 링크",
  "chapters": [
    {
      "title": "Part 1 · 0:00–30:00 — 큰 주제",
      "sections": [
        {
          "title": "00:00 — 북마크 소주제",
          "lines": [
            { "speaker": "Steven", "en": "English line...", "ko": "한글 번역..." },
            { "speaker": "Guest",   "en": "English line...", "ko": "한글 번역..." }
          ]
        },
        { "title": "04:15 — 다음 북마크", "lines": [] }
      ]
    }
  ]
}
```

- `chapters` = 파트(30분 단위) 탭 / `sections` = 파트 안 **북마크**(타임스탬프 + 소주제). 인덱스를 클릭하면 해당 구간으로 점프
- `sections` 없이 옛 형식(`lines` 바로)도 동작함 (북마크 1개로 취급)
- `speaker` 생략 가능(나레이션형이면 빈 문자열), `ko` 생략 가능

## 방법 B — 대본 직접 붙여넣기

이미 대본이 있으면 **+ 영상 추가 → 대본 붙여넣기** 탭에 그대로.
`## 소제목` 줄로 챕터를 나눌 수 있음. (인물 구분·한글은 없음)

## 학습 흐름 (앱 안에서)
1. 챕터별로 대본 읽으며 모르는 표현 **드래그 → 🖍 형광펜**
2. 다 칠했으면 **📥 단어 정리 → 단어장** 버튼
3. 단어장에서 뜻·예문 채우기
4. **Summary 작성** → 세션에서 첨삭받은 뒤 **💾 수정본 저장**
5. 챕터 다 알면 **✅ 챕터 마스터 완료**
6. 세션 탭 → 단어 시험 / Summary 리뷰 / Speaking
